Обратная польская запись (ОПЗ) — это форма записи математических и логических выражений, где операторы расположены после своих операндов. Например, выражение `(1 + 2) * 4` в ОПЗ будет выглядеть как `1 2 + 4 *`. 

Суть ОПЗ в том, что она упрощает порядок выполнения операций и убирает необходимость в скобках для определения приоритета.

**Преимущества:**
1. Нет необходимости в скобках.
2. Операции выполняются как только оператор появляется. Это позволяет выполнять вычисления "на лету" и делает код более эффективным.

**Как это работает:**
- Для преобразования обычного выражения в ОПЗ используются два стека: один для чисел и один для операторов.
- Проходишься по входному выражению: если это число, добавляешь его в стек для чисел; если это оператор, проверяешь стек с операторами и, если нужно, выталкиваешь операторы в основной стек, добавляя новый оператор затем.
- В конце все операторы из стека операторов перемещаются в основной стек.

**Пример с `(1 + 2) * 4`:**
1. Стек чисел пуст, стек операторов пуст.
2. Читаем `1`, это число, добавляем в стек чисел.
3. Читаем `+`, это оператор, добавляем в стек операторов.
4. Читаем `2`, это число, добавляем в стек чисел.
5. Читаем `)`, это скобка, выталкиваем `+` из стека операторов и два последних числа (`1` и `2`), применяем операцию и результат (`3`) обратно в стек чисел.
6. Читаем `*`, это оператор, добавляем в стек операторов.
7. Читаем `4`, это число, добавляем в стек чисел.
8. Выражение кончилось, выталкиваем `*` и два последних числа (`3` и `4`), применяем операцию и результат (`12`) — это и есть результат вычисления выражения.

После преобразования в ОПЗ вычисление производится легко и быстро, снова с использованием стека. Каждый раз, когда встречается оператор, берутся два последних числа из стека, выполняется операция, и результат кладётся обратно в стек. В конце в стеке остается одно число — результат.
